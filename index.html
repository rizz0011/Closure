Closure

Closure is the combination of function bundled together with references to it's surrounding state (the lexical enviornemt). We can also say closure give us to access an outer scope to inner scope

    Use Cases of Closure:

  • Private Variables: Closure are commonly used to create private variables and functions within an outer function,
  • preventing direct access from outside the scope.
  • Callback Functions: Closure are useful in callback patterns, where a function needs to remember and access its surrounding state after it is passed as a callback.
